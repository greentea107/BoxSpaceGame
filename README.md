# Space Battle（空间大战）
==
这是一款Android平台独占的射击型小游戏。游戏使用Android原生的API和控件编写而成，旨在展示这个API和控件除了可以用于
应用程序的开发以外也能用于游戏的编写。 

##玩法和操控
通过十字键控制屏幕中央的箭头进行八个方向的移动，并通过发射子弹消灭出现的敌人，在发射子弹的时候玩家的箭头方向会被
锁定，也可以通过“瞬移”按钮躲避敌人的包围，当敌人全部消灭后即为通关。
